# Heart-Stroke-Prediction


## Git commands

```bash
git add .

git commit -m "Updated"

git push origin main
```

## How to run?

```bash
conda create -n heart python=3.9 -y
```

```bash
conda activate heart
```

```bash
pip install -r requirements.txt
```

```bash
python app.py
```
